# AndroidMusicPlayer

This is a basic music player app project for my students.  The objective here was to introduce them to the basics of: 
* Multiple Activities
* New Views, such as ListView and Seekbar
* Media Controls
* Metadata
* Threads and Runnables
* Retreiving Files
* Working with Permissions
